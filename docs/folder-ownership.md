# Folder Ownership

## Backend

- `app/Http/Controllers` — route handlers and page controllers
- `app/Services` — business logic, orchestration, and domain services
- `app/Http/Requests` — validation and authorization rules
- `app/Models` — Eloquent models and domain entities
- `routes` — HTTP route definitions for web and API
- `database` — migrations, factories, seeders

## Frontend

- `resources/js/Pages` — page entry points for Inertia routes
- `resources/js/Components` — reusable Vue components
- `resources/js/Composables` — shared composition functions
- `resources/js/Stores` — state management helpers and stores
- `resources/js/Lib` — utility functions and shared libs
- `resources/views` — Inertia root view and Blade fallback pages
